快速幂

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n == 1:
            return x
        if n == 0:
            return 1
        if n < 0:
            return self.myPow(1/x, -n)
        return self.myPow(x, n // 2) ** 2 if n % 2 == 0 else self.myPow(x, n // 2) ** 2 * x
```
